Walkability in the US (May 2023)

My partner and I analyzed Census Walkability data using Pandas and plotted on an interactive choropleth map using GeoPandas and geojson files. We also presented graphs made with Plotly.express along with out choropleth maps on an easy-to-read web page built with Bootstrap.
